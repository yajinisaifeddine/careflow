# 🏥 CareFlow Backend

**CareFlow** is the backend system for a **Système de Gestion de Rendez-vous Médicaux (SGRM)** — a medical appointment management platform designed to optimize scheduling, patient–doctor interactions, and administrative workflows.  
It also integrates a medical AI chatbot for preliminary symptom analysis.

---

## 🚀 Overview

This backend provides secure REST APIs for:
- Patient and doctor authentication & authorization  
- Role-based access control  
- Appointment and scheduling management  
- Notification handling  
- AI-driven pre-diagnosis integration (planned)
